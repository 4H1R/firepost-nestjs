# FirePost Backend

Created using Nest JS

## Setup the Project

1. Clone the Project

```
git@github.com:4H1R/firepost-nestjs.git
```

2. Change your directory to the cloned project

```
cd firepost-nestjs
```

3. Install required dependencies

```
npm install
```

4. Setup the .env file

```
cp .env.example .env.local
```

5. Run the Project

```
npm run start:dev
```
